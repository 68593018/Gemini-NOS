# GEMINI.md

当前目录是一个空工程，用于网络操作系统平台软件相关的分析、验证和原型开发。

工作规则：

1. 使用中文回答。
2. 可以按任务逐步创建目录和文件。
3. 新建文件前先说明设计思路。
4. 代码保持简单清晰，不做过度设计。
5. 不随意引入复杂框架。
6. 每次修改后说明变更内容。
7. 编程语言主体使用 C 语言；辅助脚本、测试工具、数据处理工具可以使用 Python。
8. 文档采用工程汇报风格，优先使用“一句话结论 + 表格 + 简要说明”。
9. 不确定内容不要写成确定结论。
10. 每次代码修改并通过验证后，必须及时进行 git commit 并 git push 同步到远程仓库。
